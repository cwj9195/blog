正则 ​

  • 如果你想要在替换文本中包含原始匹配的内容，你可以使用$& | $0 来引用整个匹配的字符串，或者使用$n来引用第n个捕获组的内容。这里的n是一个从1开始的数字，代表正则表达式中的第n个捕获组。
js
// 跟vscode里的查找替换是一样的
const s = `<div class=" text-[24px] font-[600] ">{{ filterM(msg.aftSendProgress) }}<span class="text-[12px]">%</span></div>`
const r= /\sclass="(.*?)"/
// ================================================
console.log(s.replace(r,' :class="`$1`"')) // 替换 `class` `:class="`$1`"`
console.log(r.exec(s)) //找到匹配项

// ================================================
// 移动端ui设计图 ui设定是375px来做的
const s = ':class="`bg-white rounded-[14px] mx-1 mb-[10px] py-[20px] px-[16px]`">'
const r = /\[(.{1,3})px\]/
console.log(s.replace(r,'[${$1/37.5}rem]')) // 将px 替换rem
